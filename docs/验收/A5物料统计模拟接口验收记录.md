# A5、物料与统计模拟接口验收记录

日期：2026-07-16

## 验收环境

- 后端：FastAPI + SQLite 临时测试库。
- 外部接口：`A5_ADAPTER_MODE=mock`、`MATERIAL_ADAPTER_MODE=mock`。
- A5 模拟场景：`normal`，包含日报、施工异常和特殊工序数据。
- 物料模拟场景：`normal`，包含计划事件和重复事件幂等验证。

## 已验证场景

| 场景 | 预期结果 | 验证方式 |
|---|---|---|
| 小写 A5 回调头 | 正确 HMAC 被接受 | `test_a5_callback_security.py` |
| 缺失 A5 密钥 | 回调拒绝 | `test_a5_callback_security.py` |
| 重复 A5 事件 | 工单仅更新一次 | `test_a5_event_workflow.py` |
| 未匹配 A5 工单 | 事件保留为待复核 | `test_a5_event_workflow.py` |
| A5 模拟接口 | 日报、异常、工序返回稳定数据 | `test_a5_adapter.py` |
| 物料模拟接口 | 合法计划流转，重复事件不重复计量 | `test_material_external_adapter.py` |
| 数据范围 | 物料、统计和导出限制到填报单位 | `test_material_data_scope.py`、`test_statistics_data_scope.py` |
| 前端模式标识 | A5 页展示模拟或正式接口模式 | `test_frontend_integration_modes_contract.py` |

## 已知限制

- `http` 适配器保留为真实接口边界，实际字段映射和凭据须由 A5、仓储或供应商系统提供后配置。
- 非全局用户的物料外部事件尚无可安全关联的填报单位字段，因此统计中不展示该类事件计数，避免跨单位泄露。
- 前端构建仍提示现有 Element Plus 与图表包体较大；该告警不影响本次功能验证。
