## 验收环境与方法

- 后端：FastAPI + SQLite 本地数据库。

- 本地 API：服务健康检查、登录、数据分析、A5 mock 与物料 mock 接口。

## 本轮修复

联调中的辖区权限验证发现：数据分析的 `integration_status` 对 A5 已处理事件仅按上报单位统计，未沿工单和项目关联传递 `DataScope`，导致辖区用户的聚合结果可能混入其他单位的事件。

- 修复提交：`1c9dd20 fix: scope integration status by territory`
- 回归提交：`ec7b7a0 test: exclude foreign A5 events from territory scope`
- 修复方式：从 `IntegrationEvent` 关联至工单和项目后应用 `reporting_unit_scope_predicate(scope)`；测试同时创建辖区内和辖区外的已处理 A5 事件，断言统计只计入辖区内事件。

该问题已通过自动化回归，当前不构成遗留发布缺陷。
